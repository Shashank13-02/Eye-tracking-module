# NeuroScreen

**NeuroScreen** is an AI-powered, non-invasive screening system designed to assess neurodevelopmental risks (such as ASD, ADHD, and cognitive delays) during early childhood. It leverages advanced computer vision and gaze analytics to extract objective digital biomarkers from children as they interact with specially designed visual tasks.

---

## 🎯 Key Features

- **Non-Invasive Tracking**: Uses standard smartphone or laptop webcams—no specialized eye-tracking hardware required.
- **Real-Time Analysis**: Powered by Google MediaPipe Face Mesh for real-time extraction of 478 facial landmarks, including iris and eye contours.
- **Automated Digital Biomarkers**: Computes gaze metrics such as:
  - Fixation duration & saccade length (I-VT velocity-threshold algorithm).
  - Blink rate (Eye Aspect Ratio).
  - Region-of-Interest (ROI) dwell time and transition latencies.
- **Interactive Screening Tasks**: Includes age-appropriate visual tasks designed to target specific neurodevelopmental markers:
  - Face vs. Object Preference (ASD)
  - Social Scene Scanning (ASD)
  - Smooth Pursuit Tracking (Cognitive Delay)
  - Joint Attention Response (Social Communication)
  - Anti-Saccade Inhibition (ADHD)
  - Sustained Attention (ADHD)
  - Pattern Recognition (Cognitive Delay)
- **Multi-Dimensional Risk Engine**: Aggregates behavioral features to generate risk scores across multiple neurodevelopmental categories.
- **Premium Dashboard**: A beautifully designed, interactive UI built with Streamlit and Plotly for clinicians to guide screenings and review results.
- **PDF Report Generation**: Exports structured clinical screening reports summarizing the child's performance and risk indicators.

## 🛠️ Technology Stack

- **Computer Vision**: [MediaPipe](https://mediapipe.dev/) (Face Mesh) & OpenCV
- **Data Processing**: NumPy, Pandas, SciPy, Scikit-learn
- **Dashboard & Visualization**: Streamlit, Plotly, Matplotlib
- **Reporting**: fpdf2

## 🚀 Getting Started

### Prerequisites
- Python 3.8+
- Webcam

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Shashank13-02/Eye-tracking-module.git
   cd Eye-tracking-module
   ```
2. Create and activate a virtual environment (recommended):
   ```bash
   python -m venv .venv
   
   # Windows
   .\.venv\Scripts\activate
   # Linux/Mac
   source .venv/bin/activate
   ```
3. Install the dependencies:
   ```bash
   pip install -r requirements.txt
   ```

### Running the Application
Launch the interactive Streamlit dashboard:
```bash
streamlit run app.py
```
Open your browser to `http://localhost:8501` to access the screening interface.
